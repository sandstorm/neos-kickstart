# requirements 
- docker for mac


# local dev setup

- run `composer install` in `/neos`
- run `docker-compose build`
- run `docker-compose up -d`
- run `docker-compose logs -f`
- run `docker-compose stop`
- run `docker-compose down` (cleanup)
